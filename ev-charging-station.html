EV Charging Station Recommendation System

Project information

  • Category: Information Retrieval
  • Project date: Sept. 2023 - Dec. 2023

Our project aims to build a recommendation system for electric vehicle charging stations. The envisioned engine would integrate factors including access status, number and type of plugs and pricing, among other information. Given the user predefined information, such as the vehicle model, payment method or charging speed preference, our system would recommend the most feasible charging station nearby with the best charging experience.

The data sources we use for this project come from the National Renewable Energy Laboratory (NREL)'s developer network API and reviews from Google Maps. WIth feature engineering with acquired data, we would build and train our recommendation engine. Then, we propose assessing our recommendation system's accuracy by comparing its results with baseline performance with basic algorithms and human performed recommendation. The quantitative analysis of the ranking would be completed through metrics like MAP and NDCG.
